## 下载

### Docker

```
docker pull apachecn0/moralis-blog-zh
docker run -tid -p <port>:80 apachecn0/moralis-blog-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g moralis-blog-zh
moralis-blog-zh <port>
# 访问 http://localhost:{port} 查看文档
```
